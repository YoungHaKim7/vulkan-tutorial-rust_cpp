# cargo wgsl(wgsl문법확인 빠르게 하기)
- Validate wgsl in rust projects
  - https://crates.io/crates/cargo-wgsl

```
# wgsl있는 폴더에서 누르면 문법확인 빠르게 해줌
cargo wgsl
```
